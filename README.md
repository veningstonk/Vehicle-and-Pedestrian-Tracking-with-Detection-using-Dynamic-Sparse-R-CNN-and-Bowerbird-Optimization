# Vehicle-and-Pedestrian-Tracking-with-Detection-using-Dynamic-Sparse-R-CNN-and-Bowerbird-Optimization
Hybrid framework for vehicle and pedestrian tracking using Dynamic Sparse R-CNN with Bowerbird Optimization and an enhanced MobileNetV2 as backbone. Achieves robust detection in real-time across occlusions, low-light, and dense scenes. Evaluated on MOT and UA-DETRAC datasets.
